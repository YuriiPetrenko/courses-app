# Courses store app
Status: Currently in development...
### Get started

```
npm i
```

to install all dependencies

### How to run server:

Run a web server:

```
npm start
```

Open to view in browser: `http://localhost:3000`



### Technologies:

- Handlebars
- CSS (Materialize)
- JavaScript
- Node.js
- MongoDB
- Mongoose
- Nodemon
